# tja

Show random card from Oblique Strategies.

## Install

```bash
cargo install --locked --git https://github.com/coignard/tja
```

Or build manually:

```bash
git clone https://github.com/coignard/tja
cd tja
cargo build --release
sudo cp target/release/tja /usr/local/bin/
```

## Usage

```bash
tja [--lang en|ru]
```
